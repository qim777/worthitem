# 值物



> 一个专注于物品管理、订阅追踪与真实购买力换算的应用。

### 🌟 核心功能

* **📦 物品清单**：记录你拥有的生产力工具，管理保修与折旧。
* **📅 订阅管理**：一目了然所有周期性账。
* **📊 购买力对比**：将开支具象化。比如这笔订阅费 = 3杯咖啡/月，或者 2小时的工作量。

* 还有高度主题自定义与可以完全本地化与私人化
